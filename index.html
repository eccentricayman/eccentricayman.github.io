Signum Event Systems

An event sign in system that uses facial recognition in order to make signing into events a breeze. The backend makes use of Python and Clarifai in order to try and recognize a person's face from their profile and events they attend, getting more accurate with every new event that they attend utilizing this system.

Concert.io

A web app to help you get introduced to the artists performing at your concerts. The web app can query a search and use it to find an individual artist's current concerts, or concerts happening near a location. It then provides information about the concert, letting a user buy tickets, see the location and time, view the artists performing, and get a list of the artist's most popular songs to get a sense of what's going on in the concert.
